# File Copy Tool (C)

This project copies the content of one file into another file.

## Features
- opens a source file
- creates a destination file
- copies all characters from source to destination

## Concepts used
- file handling
- loops
- character processing

## Input file
Create a file named `source.txt`.

## Output file
The program creates a file named `copy.txt`.

## Compilation
gcc copyfile.c -o copyfile

## Execution
./copyfiles
